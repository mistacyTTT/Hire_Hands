Project Title: Hire Hands
Hire Hands: A Platform for Hiring Trusted Home Service Workers

Presented by:
Ishya Stacy Mwanafunzi
Umukunzi Kenzo

* Introduction (The Problem)

In many homes today, people often need help with small but important tasks such as: Plumbing repairs, Carpentry work,
Masonry jobs, Electrical fixes, Cleaning and housework and more.

However, finding reliable and skilled workers can be difficult because: There is no central place to find them,
Trust is a big issue and that People cannot easily verify skills or experience.

* Project Idea (The Solution)

This project introduces a website platform that connects:

- Skilled workers looking for jobs
- Homeowners who need services

The platform allows users to: Find trusted service providers, Hire workers easily,
Verify experience before hiring and Rate and review services.

* Who Can Use the Platform

Plumbers, Masons, Carpenters, Electricians, Painters, Housemaids / Home helpers and others who want to do freelancing.

* Clients (Customers)

Homeowners, Tenants, Small businesses.

* Worker Registration Features

Workers can create accounts where they: Upload personal details, Provide proof of experience,
Upload certificates or documents, Add skills and service categories.

This helps ensure credibility and trust.

* Hiring Features for Clients

Clients can: Search workers by skill, View worker profiles, See experience and documents,
Contact and hire workers easily.

This makes hiring fast and safe.

* Trust System (Ratings & Reviews)

After work is completed: Clients can rate workers, Clients can leave comments, Workers build reputation over time.

This creates a trustworthy system.

* Relevance

This project is very important because:Finding trusted workers is challenging, Youth unemployment remains an issue,
There is a strong culture of hiring house helpers.

* Connection to TVET Education:

This project supports TVET students because:

while Students graduate job-ready, Many TVET graduates need job opportunities. So Thi platform helps them to:

Showcase their skills, Get hired quickly, Build professional reputations.

* Conclusion

This platform:

-Solves real-life hiring problems

-Supports skilled workers

-Promotes TVET education

-Builds trust between clients and workers

It is a practical solution that can make a positive impact in the community.
